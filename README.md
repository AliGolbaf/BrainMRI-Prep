## Installation
BrainMRI-Prep requires Python 3.9 or higher. Use ([pip](https://pip.pypa.io/en/stable/)):
```shell
pip install git+https://github.com/AliGolbaf/BrainMRI-Prep
```
## Usage
Run the following command:
```shell
brainmri_prep
```
